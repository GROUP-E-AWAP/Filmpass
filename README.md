# Filmpass

This is a web application with a React frontend and a Node.js (Express) backend.

## Backend

The backend is a Node.js application using Express and PostgreSQL.

### Setup

1.  Install dependencies: `npm install`
2.  Create a `.env` file with your PostgreSQL connection details.
3.  Run the database migrations in `schema.pg.sql` and `seed.pg.sql`.
4.  Start the server: `npm start`

## Frontend

The frontend is a React application built with Vite.

### Setup

1.  Install dependencies: `npm install`
2.  Start the development server: `npm run dev`
3.  Build for production: `npm run build`
